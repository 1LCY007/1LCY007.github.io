May. 2025 - Present / Agent Systems

RepoFlow: Repository Governance for AI Coding Agents

RepoFlow is a skill-based repository governance framework for LLM coding agents such as Claude Code and Codex. It focuses on preserving repository structure, style consistency, and handoff quality during long-term agent-assisted development.

Coding Agents Claude Code Codex Agent Handoff Style Learning Code Health Review Repository Scaffold
  • Repository governance framework Designed and implemented a skill-based framework that encodes project structure, coding conventions, and review policies.
  • Agent handoff and style learning Developed mechanisms for structured handoff, shared context, and repo-local style learning to reduce drift across agent turns.
  • Code health review Built review logic for detecting patch-like naming, duplicate helpers, wrapper functions, suspicious files, and repository degradation.
