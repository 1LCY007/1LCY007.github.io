Agent Systems / RepoFlow

RepoFlow: repo-local coordination for sequential coding agents

RepoFlow is an experimental agent project developed from the BoardFlowBench repository. It studies how coding agents can pass work across turns through repository-local shared state, structured task boards, handoff records, scope rules, and automated code-health checks.

Coding Agents Repo-local State Agent Handoff Benchmarking Code Health Review Python
  • Agent Bridge Maintains task boards, handoff JSON records, review notes, and shared context for the next agent.
  • Code Health Review Detects patch-like naming, duplicate helper patterns, suspicious files, wrapper functions, and other AI-generated code risks.
  • Benchmark Checks Scores task completion, scope control, handoff completeness, repository hygiene, and board consistency.
